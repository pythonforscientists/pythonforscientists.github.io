Python for Scientists - Instructor Resources

Instructor resources are available to qualified instructors by request. Instructor resources include an instructor's edition of the Python for Scientists textbook, which contains an answer key to all exercises, annotations in the text, objective breakdowns, POGIL answer documents, and lab guidance.

To gain access to instructor resources, send an email to pythonforscientists@protonmail.com. We'll get back to you with ways to verify instructor status.
