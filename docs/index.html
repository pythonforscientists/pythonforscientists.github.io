Welcome to the website for the Python for Scientists textbook! This website has materials for students and instructors alike.

Downloads

If you are looking to download the textbook, click here. This link will take you to the releases page for the Python for Scientists textbook, where you can download the latest version of the textbook as a PDF file.

Careful! If your instructor gave you a custom version of the textbook, you should use that version instead. It might have changes to material or exercises that your instructor made.

Contribute
